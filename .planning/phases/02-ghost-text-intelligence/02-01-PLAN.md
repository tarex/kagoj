  <files>src/hooks/useAISuggestion.ts</files>

  <action>
    Create `src/hooks/useAISuggestion.ts`:

    1. Import `LRUCache` from `@/lib/lru-cache`, React hooks

    2. Module-level constants:
       - `AI_CACHE_SIZE = 50`
       - `AI_RATE_LIMIT_MS = 2000` (max 1 request per 2 seconds)
       - `AI_TRIGGER_DELAY_MS = 500` (wait 500ms after word boundary)

    3. Module-level singletons (shared across hook instances):
       - `const suggestionCache = new LRUCache<string, string>(AI_CACHE_SIZE)`
       - `let lastRequestTime = 0`

    4. Hook signature: `useAISuggestion(isBanglaMode: boolean)`
       Returns: `{ aiSuggestion: string, isLoadingAI: boolean, requestAISuggestion: (cursorContext: string) => void, clearAISuggestion: () => void }`

    5. Internal state: `aiSuggestion` (string), `isLoadingAI` (boolean)

    6. `requestAISuggestion(cursorContext: string)`:
       - Trim cursorContext to last 200 chars
       - Generate cache key from last 100 chars of trimmed context
       - Check cache — if hit, set aiSuggestion from cache, return
       - Check rate limit — if `Date.now() - lastRequestTime < AI_RATE_LIMIT_MS`, skip
       - If not bangla mode, clear and return
       - Set `isLoadingAI = true`, update `lastRequestTime = Date.now()`
       - Use AbortController for request cancellation
       - `fetch('/api/suggestions', { method: 'POST', body: JSON.stringify({ text: '', mode: 'ghost', cursorContext }), signal })`
       - On success: extract `suggestion`, store in cache, set `aiSuggestion`
       - On error/abort: set aiSuggestion to '', log error only if not abort
       - Always set `isLoadingAI = false` in finally

    7. Cleanup: abort in-flight request on unmount

    8. `clearAISuggestion()`: sets aiSuggestion to ''
  </action>
