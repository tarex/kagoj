   - Use `phoneticDistance` instead of `levenshteinDistance` for ranking candidates.
   - Return top candidate sorted by phoneticDistance (lowest first).

   - Compare prevText and newText to find the edit region (first differing char index from start, first differing from end).

   - Return only errors whose ranges do NOT overlap with the edit region.
