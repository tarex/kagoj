Create `src/lib/trie.ts` with a custom lightweight trie class (~100 lines). No external dependencies.

TrieNode structure:
```typescript
interface TrieNode {
  children: Map<string, TrieNode>;
  isEndOfWord: boolean;
  frequency: number;
  word: string | null; // Store full word at leaf for easy retrieval
}
```

BanglaTrie class with these methods:
- `insert(word: string, frequency?: number): void` — Insert a word, optionally with frequency. If word exists, update frequency (add, don't replace).
- `search(word: string): boolean` — Exact word lookup. Return true if word exists in trie.
- `hasPrefix(prefix: string): boolean` — Return true if any word starts with prefix.
- `getSuggestions(prefix: string, limit?: number): string[]` — Return up to `limit` (default 10) words starting with prefix, sorted by frequency descending then by word length ascending. Use DFS traversal collecting matches.
- `getFrequency(word: string): number` — Return frequency for exact word, or 0 if not found.
- `setFrequency(word: string, frequency: number): void` — Update frequency for existing word.
- `size(): number` — Return total word count. Track this as a private counter incremented on insert of new words, not by traversal.
- `getAllWords(): string[]` — Return all words (for serialization/debug). Use DFS.
- `delete(word: string): boolean` — Soft delete by setting isEndOfWord=false. Return true if word existed.

Export the class as `BanglaTrie` and also export the `TrieNode` interface.

Key implementation details:
- Each character in a Bangla word is a key in the children Map (Bangla Unicode characters, not bytes)
- Use Map not object for children — Bangla characters as keys work better with Map
- DFS for getSuggestions should use a priority approach: collect all matches, sort by frequency desc then length asc, return top N
- TypeScript strict mode, no `any`

Word generation strategy:
1. Start with ALL words from existing `bangla-dictionary.ts` and `bangla-words-extended.ts`
2. For each of the 500+ most common verb roots, generate 5 conjugations: present simple (করে), present continuous (করছে), past simple (করেছে/করল), future (করবে), and imperative (করুন/করো)
3. Add comprehensive noun categories: common objects, places, animals, plants, food, body parts, kinship, occupations, emotions, abstract concepts, academic terms, technology terms
4. Add adjectives: descriptive (বড়, ছোট, সুন্দর...), colors, qualities, states
5. Add all postpositions, conjunctions, interjections
6. Add number words (cardinal, ordinal), day names, month names (both Bengali and Gregorian), season names

Target: 50,000+ unique words minimum. This is a large file — that is expected and intentional per the decision to ship as a single flat file.

The file should ONLY export the array. No functions, no classes. Pure data.

Ensure no duplicate words in the array. The executor should programmatically verify uniqueness before finalizing.

IMPORTANT: Every word must be valid Bangla Unicode (range \u0980-\u09FF plus hasanta \u09CD for conjuncts). No English, no mixed scripts.
