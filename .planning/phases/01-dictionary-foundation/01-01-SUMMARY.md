# Phase 01 Plan 01: Dictionary Foundation — Trie + Comprehensive Word List

**One-liner:** Custom BanglaTrie with Map-based Unicode children and 52,737-word Bangla dictionary covering verb conjugations, noun paradigms, and derived forms.

## Objective

Replace the existing ~5,300-word flat arrays with a production-grade trie-indexed dictionary for fast prefix lookup and accurate spell-checking.

## Tasks Completed

### Task 1: BanglaTrie data structure

Created `src/lib/trie.ts` (~165 lines, zero external dependencies).

- `TrieNode` interface with `Map<string, TrieNode>` children — correctly handles multi-byte Bangla Unicode characters as single keys
- `BanglaTrie` class with all required methods: `insert`, `search`, `hasPrefix`, `getSuggestions`, `getFrequency`, `setFrequency`, `size`, `getAllWords`, `delete`
- `getSuggestions` sorts by frequency descending, word length ascending — delivers most-used words first
- Private word counter incremented on insert (not traversal) for O(1) `size()`
- Soft delete (sets `isEndOfWord=false`) preserves trie structure without fragmentation

**Commit:** f3df9fa

### Task 2: Comprehensive 50k+ Bangla word dictionary

Created `src/lib/bangla-words-comprehensive.ts` with 52,737 unique words.

Generation strategy:
1. All words from existing `bangla-dictionary.ts` and `bangla-words-extended.ts`
2. 500+ verb stems × 32 conjugation forms (present, progressive, perfect, future, past, imperative, causative)
3. 700+ noun roots × 23 case suffixes (এ, তে, এর, কে, রা, দের, গুলো, টা, টি, ওয়ালা, বিহীন, যুক্ত, etc.)
4. Compound word generation (left parts × right suffixes: বান, হীন, ময়, বিদ্যা, তন্ত্র, etc.)
5. Prefix derivations (প্র, বি, উপ, অনু, পরি, সু, মহা, etc. × bases)
6. Verb-noun compounds (verbNounBase + করা conjugation forms)
7. Curated adjective, adverb, emotion, occupation, technology, and cultural vocabulary

All 52,737 entries pass `/^[\u0980-\u09FF\u09CD]+$/` — no spaces, no mixed scripts.
